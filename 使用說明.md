# 翰林業務教育訓練平台（純靜態網站）

這是一般的 HTML + CSS + JavaScript 純靜態網站，**不需要任何伺服器或啟動程式**。

## 怎麼開
直接雙擊 **`index.html`**，用瀏覽器（Chrome / Edge 建議）打開即可。

## 怎麼上線（之後若要放到網路上）
把整個 `site` 資料夾的內容，丟到任何靜態空間都能跑，例如：
- 公司內部網站空間
- GitHub Pages、Netlify、Cloudflare Pages 等免費靜態託管

## 平台怎麼運作
- **首頁＝學習地圖**（`index.html`）：每個「專案」是一個學習區塊。目前有「會員中心」。
- **進入區塊後，依序完成 3 步**（`course.html`）：
  1. 看簡報（翰林會員中心 PDF，可在頁面內翻頁）
  2. 看影片（推廣與導覽教戰手冊）
  3. 作測驗（外連 Google 表單，會開新分頁）
- **依序解鎖**：沒完成上一步，下一步是鎖住的。每步看完按「我已完成」才解鎖下一步。
- 進度記在這台電腦的瀏覽器（localStorage），下次打開會接續。

## 檔案結構
```
site/
├─ index.html          首頁（學習地圖）
├─ course.html         課程頁（三步驟）
├─ css/style.css       樣式
├─ js/
│   ├─ courses.js      ★ 課程資料設定（要新增專案改這裡）
│   ├─ progress.js     學習進度
│   ├─ home.js         首頁邏輯
│   └─ course.js       課程頁邏輯
└─ assets/會員中心/     簡報PDF、影片
```

## 之後要新增別的專案區塊
打開 `js/courses.js`，照「會員中心」那一段複製一份，改成新的標題、PDF、影片、測驗連結，`available` 設成 `true` 就會上架。新素材放到 `assets/（你的專案名）/`。

> 簡報原始檔（PPTX）已轉成 PDF 放在 `assets/會員中心/`，方便在網頁直接觀看。
> 之後簡報若更新，重新轉成 PDF 覆蓋同名檔案即可。
